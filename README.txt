Following libraries are used
1.numpy
2.scikit-learn
3.itertools
4.pandas
5.Os
6.scikit-image
7.opencv


To run the project following steps should be followed
1.run feature-extraction.py after which an excel will be generated with features and labels
2.run machine learning.py which gives output in terms of accuracy on the console and a confusion matrix is also generated for gradient boosting.
